# Strike Log — 포트폴리오

## 1) 포트폴리오 소개
**볼링 동호인·취미 볼러를 위한 기록 & 클럽 커뮤니티 앱 개발**
- 서비스 카테고리: 스포츠/취미 기록 + 커뮤니티(SNS) 모바일 앱
- 메인 타깃: 볼링 동호회(클럽) 멤버 및 개인 볼러
- 개인 점수 기록·통계에서 나아가, **클럽이 정기전(정모)을 열어 함께 경기하고 기록이 쌓이는 커뮤니티**로 확장. App Store·Google Play 동시 출시.

## 2) 작업 범위
**1인 풀스택 개발 — 기획부터 스토어 출시까지 전 과정 단독 수행**
- 서비스 기획 · UX/UI 디자인
- **Front-end**: Flutter 모바일 앱 개발
- **Back-end**: NestJS API 서버 구축
- **실시간 서버**: Socket.IO 기반 게임룸(멀티플레이) 게이트웨이
- **DB**: MySQL 스키마 설계 및 TypeORM 마이그레이션
- 인증/푸시/결제성(광고) 등 외부 연동, 인프라·배포·CI, 스토어 심사 대응

**지원 환경**: Android · iOS (Flutter 크로스플랫폼)

## 3) 주요 업무
- **볼링 점수 기록 엔진**: 10프레임·스트라이크/스페어 보너스 룩어헤드·10프레임 특수 룰·누적 점수 실시간 계산
- **게임 모드 4종**: 개인 / 시리즈(합계·평균) / 클럽(실시간 공유) / 내기(핸디캡)
- **실시간 멀티플레이 게임룸**: 방 코드 참여, 점수 실시간 동기화(Socket.IO), 내기 자동 핸디캡·순위 집계
- **클럽 커뮤니티 + 3단계 권한(RBAC)**: 클럽장/운영진/일반멤버, 가입 승인제, 멤버 관리(임명·해제·이양·추방)
- **정기전(모임) 기능**: 이벤트 생성, 멤버 자가 참석(RSVP), **레인 자동 배치(랜덤·에버리지 균형·팀전)**, 순위·평균 자동 집계, 결과 이미지 공유
- **통계·랭킹·챌린지/배지**: 종합·개인·클럽 에버리지, 월별 통계, 출석 스트릭, 클럽 랭킹
- **계정/인증**: 이메일·비밀번호 로그인 + JWT 자동 로그인, **이메일 OTP 인증**(회원가입·비밀번호 재설정)
- **푸시 알림(FCM)**: 가입 신청·공지·체험 만료 등, iOS ATT 동의 흐름
- **수익화·운영**: AdMob 광고, 클럽 구독/체험 게이팅, 공지/1:1 문의/약관 관리
- **주요 페이지**: 홈 대시보드, 점수 입력, 경기 요약/공유, 기록 목록, 클럽 허브, 정기전 목록/상세, 랭킹, 멤버 관리, 로그인/회원가입

## 4) 주안점
- **점수 계산의 정확성**: 보너스 룩어헤드·10프레임 예외 처리, 프레임 재입력 시 데이터 손상 방지 및 반응형 누적 계산
- **실시간 동기화 정합성**: Socket.IO 게임룸에서 다중 참가자 점수 동기화, `room_id` 기반 기록 그룹화, 구독 상태 기반 접근 게이팅
- **권한 모델 안전성**: 2단계 → 3단계 권한 확장 시 기존 데이터를 **마이그레이션으로 무손실 변환**, 역할 위계 기반 액션 제어, 소유권 이양은 트랜잭션으로 원자 처리
- **개인정보·인증 보안**: 이메일 OTP를 5분 유효·**1회성 소비(단일 DELETE로 TOCTOU 방지)**, 사용자 존재 노출을 막는 통일 에러 응답, JWT 세션·401 자동 로그아웃 가드
- **비침습 확장 설계**: 정기전 집계를 기존 게임 저장 경로에 `event_id` 태깅만으로 연동해 실시간 로직 무변경
- **출시·배포 효율**: iOS/Android 양대 스토어 출시 파이프라인 + **Shorebird OTA 코드 푸시**로 Dart 수정분을 재심사 없이 즉시 배포
