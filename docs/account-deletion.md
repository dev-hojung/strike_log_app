# 계정 삭제 안내 (회원 탈퇴)

스트라이크 로그는 사용자가 언제든지 계정과 관련 데이터를 영구 삭제할 수 있도록 두 가지 경로를 제공합니다.

---

## 1. 앱 내에서 직접 탈퇴 (권장)

1. 스트라이크 로그 앱 실행 → 로그인
2. 하단 네비게이션 **프로필** 탭
3. **계정 설정** → **계정 관리 → 회원 탈퇴**
4. 확인 다이얼로그에서 **탈퇴하기** 선택
5. 즉시 처리되며 로그인 화면으로 복귀

---

## 2. 이메일로 요청

앱에 접근이 어렵거나 다른 사유로 직접 처리할 수 없는 경우:

- 받는 사람: **dev.hojung@gmail.com**
- 제목: `[Strike Log] 회원 탈퇴 요청`
- 본문: 가입 시 사용한 **이메일 주소**

운영자가 본인 확인 후 7일 이내에 처리합니다.

---

## 삭제되는 데이터

탈퇴 시 다음 데이터가 **즉시·영구 삭제**되며 복구할 수 없습니다:

- 계정 정보 (이메일, 비밀번호, 닉네임, 전화번호, 프로필 이미지)
- 게임 및 시리즈 기록 (모든 프레임 데이터 포함)
- 클럽 멤버십 및 가입/생성 신청 내역
- 획득한 배지와 출석 기록(streak)
- 알림 이력 및 푸시 토큰

---

## 보관 데이터 (있는 경우)

현재 스트라이크 로그는 **결제 거래 내역을 보관하지 않습니다**. 무료 앱이며 결제 시스템이 없습니다.

향후 유료 기능 도입 시, 「전자상거래 등에서의 소비자보호에 관한 법률」에 따라 관련 거래 기록은 법정 기간(최대 5년) 동안 분리 보관됩니다. 그 외 모든 식별 가능한 개인정보는 즉시 삭제됩니다.
