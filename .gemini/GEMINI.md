# Strike Log App - Gemini Instructions

1. **API 관련 작업 컨텍스트 (필수 참조)**
   - 앱(Flutter) 개발 중 백엔드 API 연동, 수정, 분석이 필요한 경우 **반드시 `~/develop/strike_log_api` 디렉토리(NestJS 프로젝트)를 참조**하여 함께 분석하고 작업한다.
   - 앱에서 새로운 API를 호출하거나 기존 엔드포인트 변경이 필요한 경우, 해당 백엔드 프로젝트의 코드를 함께 검토하고 필요시 수정안을 제시한다.

2. **디자인 작업 컨텍스트**
   - 디자인 관련 작업시에는 반드시 **FIGMA MCP**를 활용한다.
